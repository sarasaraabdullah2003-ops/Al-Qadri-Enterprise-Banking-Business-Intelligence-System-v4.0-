# Al-Qadri-Enterprise-Banking-Business-Intelligence-System-v4.0-
​A comprehensive Management Information System (MIS) designed to simulate real-world banking operations. This project integrates advanced Cybersecurity protocols with Business Intelligence (BI) tools to provide a secure and data-driven financial environment.
# 🏦 Al-Qadri Enterprise Banking Solution (v4.0)

نظام بنكي متكامل لإدارة العمليات المصرفية، مصمم لخدمة أهداف نظم المعلومات الإدارية (MIS).

## 🚀 المميزات الرئيسية:
- **Security:** نظام تشفير بيانات مخصص مع تحقق بخطوتين (2FA).
- **Business Intelligence:** لوحة تحكم رسومية لتحليل بيانات العملاء (Matplotlib).
- **Risk Management:** ميزات تجميد الحسابات وتصنيف العملاء آلياً (VIP/Gold).
- **Auditing:** سجل كامل للعمليات (Audit Trail) لمراقبة النشاط المالي.
- **Continuity:** نظام نسخ احتياطي آلي للبيانات.

## 🛠 التكنولوجيا المستخدمة:
- Python (Tkinter, Matplotlib, OS, Random).
- نظام ملفات مشفر (.dat).
